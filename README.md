# Student Registration Form - Experiment 1.1.1

## 1. Aim
To create a student registration form with HTML5 validation for name, email, and age inputs.

## 2. Objective
* Design a responsive form using semantic HTML5.
* Implement client-side validation using HTML5 attributes.
* Style form elements with CSS3.
* Handle validation feedback visually.

## 3. Tech Stack
* **Frontend:** HTML5, CSS3
* **Tools:** VS Code, GitHub

## 4. Key Features (Validation)
* **Name:** Required field with a pattern constraint (minimum 3 letters).
* **Email:** HTML5 `type="email"` for automatic format checking.
* **Age:** Numerical input with `min="17"` and `max="50"` constraints.
* **Visual Feedback:** Uses CSS `:invalid` and `:valid` pseudo-classes to highlight fields.

## 5. How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser or use the **Live Server** extension in VS Code.
